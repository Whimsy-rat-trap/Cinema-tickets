# Кинотеатр: бронирование билетов

Одностраничное приложение для резервирования билетов в кинотеатр. Пользователь может просматривать доступные даты (неделя до и неделя после текущей даты), выбирать сеанс (с 10:00 до 20:00 каждые 2 часа) и бронировать места. Данные сохраняются в localStorage, при перезагрузке страницы состояние восстанавливается. Для архивных сеансов (дата и время раньше текущего) просмотр доступен, но бронирование заблокировано.

## Функциональные требования (из тестового задания)

- Интерфейс отображает доступные даты для бронирования.
- Для выбранной даты показываются сеансы (6 сеансов в день: 10:00, 12:00, 14:00, 16:00, 18:00, 20:00).
- Отображение свободных и забронированных мест.
- Если выбранная дата и сеанс меньше текущего времени и даты — это архивные данные, их можно только просматривать, без возможности изменения.
- Данные хранятся в localStorage, при перезагрузке страницы считываются оттуда.
- Глубина архива: одна неделя до текущей даты.
- Максимальный период бронирования: одна неделя от текущей даты.
- Кроссбраузерная вёрстка (поддержка IE 10+).
- Реализация на JavaScript / jQuery / HTML5 / CSS3 (в проекте используется React + TypeScript, что соответствует современному стеку, но задача была расширена).

## Дополнительные библиотеки и технологии

- **React** – построение пользовательского интерфейса.
- **TypeScript** – типизация кода.
- **Redux Toolkit** – управление состоянием (бронирования).
- **D3.js** – создание сетки мест (таблицы бронирования).
- **animejs** – анимации в стиле neobrutalism (появление элементов, эффекты наведения и клика).
- **Webpack** – сборка проекта.
- **ESLint** – проверка качества кода.

## Внешний вид и стиль

Проект выполнен в стиле neobrutalism: яркие цвета, жирные рамки, массивные тени, динамичные анимации. Фон представляет собой три движущихся слоя с колонками розовых кругов, создающих эффект переливающегося градиента.

## Установка и запуск

### Предварительные требования

- Node.js версии 14 или выше.
- Браузер с поддержкой ES6

### Локальная разработка

# Клонирование репозитория
```bash
git clone https://github.com/Whimsy-rat-trap/Cinema-tickets.git
```

# Переход в директорию проекта
```bash
cd cinema-tickets
```

# Установка зависимостей
```bash
npm install
```

# Запуск в режиме разработки
```bash
npm start
```

# Сборка для production
```bash
npm run build
```

## Скрипты

- `npm start` – запуск dev-сервера с горячей перезагрузкой.
- `npm run build` – сборка проекта в папку `dist`.
- `npm run lint` – проверка кода линтером.
